About HR Web Application

🏢 Application Overview

The HR Web Application is a comprehensive employee management system built with modern Java technologies including JDBC, HikariCP Connection Pool, and PostgreSQL database integration.

This application follows the MVC (Model-View-Controller) architecture pattern and utilizes the latest web technologies for a seamless user experience.

🎯 Key Features

  • 📋 Employee Listing & Management
  • 🔍 Advanced Search Capabilities
  • ✏️ Employee Record Updates
  • 💰 Salary Management System
  • 🔐 Role-based Access Control

👥 User Roles & Permissions

👤 HR Staff

  • View all employee records
  • Search employees by ID
  • Access employee information
  • Generate employee reports

👨‍💼 HR Manager

  • All HR Staff permissions
  • Update employee records
  • Delete employee records
  • Process salary increments
  • Manage employee data
